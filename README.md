# DiAgnostiK - Projekt
## Buch scan tool

### Prerequisites

- `pipx install ocrmypdf`
- `apt-get install tesseract-ocr-deu`

    - for ocr: `ocrmypdf -l deu`

- `apt-get install unpaper`

    - for ocr: `ocrmypdf --clean`

- `apt install ghostscript`

    - for `gs`

- `apt install poppler`
    
    - for `pdfimages`
    - for `pdfunite`

- `apt install imagemagick`

    - for `convert`
